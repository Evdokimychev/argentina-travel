# ArgentinaTravel

Современный туристический сайт о путешествиях по Аргентине.

## Технологии

- **Next.js 15** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS 4**

## Страницы

| Страница | URL | Описание |
|----------|-----|----------|
| Главная | `/` | Hero, направления, туры, отзывы, блог |
| Каталог туров | `/tours` | Все туры с фильтрами по региону и сложности |
| Страница тура | `/tours/[slug]` | Детальное описание тура |
| Блог | `/blog` | Статьи о путешествиях |
| Статья блога | `/blog/[slug]` | Полный текст статьи |
| Контакты | `/contacts` | Форма обратной связи и контактная информация |

## Запуск локально

### Требования

- [Node.js](https://nodejs.org/) 18.18 или новее
- npm (устанавливается вместе с Node.js)

### Установка и запуск

```bash
cd ~/Projects/argentina-travel
npm install
npm run dev
```

Откройте в браузере: [http://localhost:3000](http://localhost:3000)

### Другие команды

```bash
npm run build   # Сборка для production
npm run start   # Запуск production-сервера
npm run lint    # Проверка ESLint
```

## Структура проекта

```
src/
├── app/              # Страницы (App Router)
│   ├── page.tsx      # Главная
│   ├── tours/        # Каталог и страницы туров
│   ├── blog/         # Блог
│   └── contacts/     # Контакты
├── components/       # React-компоненты
├── data/             # Данные туров, блога, отзывов
└── types/            # TypeScript-типы
```
