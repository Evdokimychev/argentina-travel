## Легенда приоритетов
`P0` — ломает доверие/индексацию сейчас · `P1` — важно · `P2` — желательно · `P3` — косметика.
Пометка `⚠️` — факт требует ручной сверки перед публикацией.

## Главные P0 (кратко)
- `G-01` непереведённые i18n-ключи (`home.hero.*`, `footer.*`) — видны пользователю, ломают H1/футер.
- `G-02` canonical/OG ведут на технический Vercel-домен.
- Гражданство (`immigration/grazhdanstvo`) описывает устаревший судебный порядок — с DNU 366/2025 это административная процедура DNM.
- `/about` показывает служебную Design System вместо страницы о проекте.
- Каталог туров на 2/3 состоит из Tripster-импорта (Бразилия/Парагвай), названного «авторскими турами».
- Фейковые отзывы/аватары и противоречивая статистика организатора.
