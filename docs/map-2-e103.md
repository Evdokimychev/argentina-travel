# E103 — Карта Аргентины 2.0

## Что добавлено

- Публичная страница `/map` — единый hub интерактивной карты страны.
- Слои (переключаются без перезагрузки): **туры**, **места**, **регионы**, **маршруты**.
- Синхронизация состояния с URL: `?layer=...&city=...&category=...&highlight=...`.
- Мобильный split view: карта сверху, список снизу; на десктопе — список слева, карта справа.
- API `GET /api/map/layers` — агрегированные точки туров/мест и маршруты с опциональным `bbox`.
- Статический GeoJSON границ провинций: `src/data/argentina-regions.geojson`.
- SEO: `generateMetadata` для `/map`, structured data `ItemList` для рекомендуемых туров.
- CDN helper `src/lib/cdn-cache.ts` для кэширования tile-adjacent данных.

## Параметры URL

- `layer` — список слоёв через запятую: `tours`, `places`, `regions`, `routes` (по умолчанию `tours`).
- `city` — фильтр по городу/региону (подстрока).
- `category` — категория места (slug категории).
- `highlight` — подсветка региона на карте (deep link из каталога туров: `/map?highlight=patagonia`).

## API `/api/map/layers`

Query-параметры:

- `layer` — как в URL страницы.
- `bbox` — `minLat,minLng,maxLat,maxLng` (опционально).
- `city`, `category` — фильтры.
- `limit` — лимит точек (по умолчанию 200, максимум 500).

Ответ:

```json
{
  "tours": [],
  "places": [],
  "routes": [],
  "totals": { "tours": 0, "places": 0, "routes": 0 }
}
```

Заголовок `Cache-Control`: `public, s-maxage=120, stale-while-revalidate=600`.

## Поведение слоёв

- **Туры / места** — кластеризация при малом зуме (логика из `PlacesCatalogMap`).
- **Регионы** — GeoJSON-полигоны с hover и подсветкой по `highlight`.
- **Маршруты** — полилинии и стартовые точки туров с seed-маршрутами из `src/data/tour-routes.ts`.

## Client/server boundary

- Leaflet и GeoJSON импортируются только в client components (`"use client"` + `dynamic(..., { ssr: false })`).
- Серверные репозитории (`map-layers-server`, `marketplace-tours-server`, `places-repository`) не попадают в client bundle.
- `next/headers` не используется в клиентских модулях карты.

## Связь с существующим кодом

- Переиспользованы: `ToursCatalogMap` (маркеры/попапы), `PlacesCatalogMap` (кластеризация), `tour-map.ts`, `argentina-cities.ts`.
- Deep link из `/tours`: `/map?highlight=<region-slug>`.
